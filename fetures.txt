Security related Features

1.All user and clinet data in database is in encripted format which  is no one decrypt

2.Every Request come from web application and from apps (android and IOS ) is served  with auth  token only 
  Without auth token data will not move from server to  web, android and IOS app and viceversa

3.Email notication will send if   user perform any securilty realed action
  These action will be like this  change password , forgot password , evey login , 2FA auth failed,
  
4.IP Address Tracking with Request and if any suspect activity found that IP address
   Will block

5.Activity log will maintain for all request
